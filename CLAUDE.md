# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Overview

This is a documentation repository focused on developer resources and programming guides. The primary content is bilingual (Chinese/English) technical documentation for beginner developers.

## Repository Structure

- **programming_beginer_guides.md** - Main documentation file containing curated developer resources including:
  - Hardware recommendations (MacBook Air M4, Framework laptops)
  - Programming languages (Rust, Python, Java, TypeScript, Go)
  - Development tools and IDEs (IntelliJ IDEA, VS Code, Eclipse, Neovim)
  - Frameworks and libraries (Spring Boot)
  - Version control and package management (Git, SDKMAN, uv)
  - Learning resources and communities
  - Open source concepts and type hinting practices

## Content Guidelines

### Language and Format
- Content is bilingual: Chinese (primary) with English terms/titles
- Use Markdown formatting consistently
- Maintain the emoji-based section headers (🖥️, 💻, 🛠️, 🚀, etc.)

### Documentation Structure
Each entry follows this pattern:
```
**Tool/Technology Name** (optional URL or reference)

- **指导建议**: [Chinese guidance text]
```

### Adding New Resources
When adding new tools, languages, or resources:
1. Place them in the appropriate section
2. Include the tool/technology name in bold with optional URL
3. Provide practical guidance in Chinese under "指导建议"
4. Focus on actionable advice for beginners (when to use, how to start, key features)
5. Keep descriptions concise (1-3 sentences)

### Version Control
- This is a git repository on the `main` branch
- Commits should clearly describe documentation additions/changes
- Example: "Add first ver of programming beginer guides"
