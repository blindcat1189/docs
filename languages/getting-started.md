# 编程语言 / Programming Languages

**Rust** (rust-lang.org)

- **指导建议**: 专注于安全性和性能的系统编程语言。适合底层开发、CLI 工具和 WebAssembly。

**Python** (python.org)

- **指导建议**: 多用途语言，适用于 Web 开发、数据科学、自动化和 AI/ML。从官方文档和教程开始学习。

**Java** (Oracle 文档)

- **指导建议**: 企业级语言。重点学习 Java 8+ 的特性，如日期时间 API、Lambda 表达式和流式处理。

**TypeScript** (typescriptlang.org)

- **指导建议**: 具有类型安全的 JavaScript。大型前端项目必备。先学习类型提示和接口。

**Go/Golang**

- **指导建议**: 简洁、快速的后端服务和云应用语言。并发支持出色。
