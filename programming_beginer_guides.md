# Developer Resources Guide / 开发者资源指南

### 🖥️ 硬件与笔记本电脑

**MacBook Air M4** - 苹果最新的轻薄笔记本，搭载 M4 芯片

- **指导建议**: 开发工作的优秀选择，电池续航长，性能出色。认真的开发工作建议选择 16GB 以上内存。

**Framework 笔记本** - 模块化、可维修的笔记本电脑

- **指导建议**: 适合重视可维修性和定制化的开发者。对开源硬件友好。

---

### 💻 编程语言

**Rust** (rust-lang.org)

- **指导建议**: 专注于安全性和性能的系统编程语言。适合底层开发、CLI 工具和 WebAssembly。

**Python** (python.org)

- **指导建议**: 多用途语言，适用于 Web 开发、数据科学、自动化和 AI/ML。从官方文档和教程开始学习。

**Java** (Oracle 文档)

- **指导建议**: 企业级语言。重点学习 Java 8+ 的特性，如日期时间 API、Lambda 表达式和流式处理。

**TypeScript** (typescriptlang.org)

- **指导建议**: 具有类型安全的 JavaScript。大型前端项目必备。先学习类型提示和接口。

**Go/Golang**

- **指导建议**: 简洁、快速的后端服务和云应用语言。并发支持出色。

---

### 🛠️ 开发工具与 IDE

**IntelliJ IDEA**

- **指导建议**: Java/JVM 语言的高级 IDE。学习时使用社区版，需要高级功能时升级。

**Visual Studio Code** (code.visualstudio.com)

- **指导建议**: 轻量级、可扩展的编辑器。安装特定语言扩展以获得最佳体验。

**Eclipse IDE** (eclipseide.org)

- **指导建议**: 免费开源 IDE，主要用于 Java。适合企业级 Java 开发。

**Neovim** / **Vim** (neovim.io, vim.org)

- **指导建议**: 面向高级用户的终端编辑器。学习曲线陡峭，但掌握后效率极高。

---

### 🚀 框架与库

**Spring Boot** (spring.io/projects/spring-boot)

- **指导建议**: 用于构建生产就绪应用的 Java 框架。使用 Spring Initializr 快速搭建项目。

---

### 📦 版本控制与包管理

**Git** (git-scm.com)

- **指导建议**: 必备的版本控制系统。学习基本命令：clone、commit、push、pull、branch、merge。

**SDKMAN** (sdkman.io)

- **指导建议**: Java 版本和 SDK 管理器。轻松切换 Java 版本并安装构建工具。

**uv** (astral-sh/uv)

- **指导建议**: 快速的 Python 包安装和解析器。性能更好的 pip 现代替代品。

---

### 📚 学习资源与社区

**Stack Overflow** (stackoverflow.com/questions)

- **指导建议**: 编程问答平台。提问前先搜索；遵循发帖指南以获得最佳回复。

**Medium** (medium.com)

- **指导建议**: 技术文章和教程平台。关注与你技术栈相关的标签。

**Markdown 指南** (markdownguide.org)

- **指导建议**: 学习 Markdown 语法，用于文档、README 文件和技术写作。

**搜索技巧**: 使用"XXX tutorial"或"head on XXX"等搜索模式查找新手指南和入门资源。

---

### 🌟 概念与理念

**开源软件**

- **指导建议**: 学习阅读、贡献和利用开源项目。从 GitHub 上的 good first issues 开始。

**类型提示**

- **指导建议**: 为代码添加类型注解（Python、TypeScript）。提高代码质量，及早发现错误。
